> 更新日期：2026-06-04
> 版本：V1.0

## 1. 学生管理

### POST /api/v1/students

创建学生档案。

**请求体：**
```json
{"name": "张三", "major": "人工智能", "target_course": "机器学习基础"}
```

**响应：**
```json
{"id": 1, "name": "张三", "major": "人工智能", "target_course": "机器学习基础", "profile": null}
```

### GET /api/v1/students

获取全部学生列表，用于侧边栏学生选择器和教师看板。

**响应：**
```json
[
  {
    "id": 1,
    "name": "张三",
    "major": "人工智能",
    "target_course": "机器学习基础",
    "learning_goal": "项目实战",
    "prerequisite_level": "零基础",
    "weak_points": ["线性回归", "模型评估"]
  }
]
```

## 2. 多智能体学习管线

### POST /api/v1/chat/profile-build

核心端点。一次性运行 6 个智能体，返回完整个性化学习方案。

**请求体：**
```json
{"student_id": 1, "message": "我是零基础，想通过项目实战学会机器学习"}
```

**响应：**
```json
{
  "student": {"id": 1, "name": "张三", "profile": {"major_background": "...", "learning_goal": "...", ...}},
  "diagnosis": {"current_stage": "...", "priority_modules": [...], "recommended_strategy": "...", ...},
  "resources": [{"resource_type": "lesson_note", "title": "...", "content": "...", "source_refs": [...]}],
  "study_plan": [{"stage": "...", "objectives": [...], "recommended_resources": [...], "practice_task": "..."}],
  "traces": [{"agent_name": "...", "input_summary": "...", "output_summary": "..."}],
  "recommendation_summary": "..."
}
```

### POST /api/v1/chat/profile-build/stream

SSE 流式版本。返回 `text/event-stream`，事件顺序：

```
event: profile
data: {"student": {...}}

event: diagnosis
data: {"current_stage": "...", ...}

event: traces
data: [{"agent_name": "...", ...}]

event: resource
data: {"resource_type": "lesson_note", ...}

...

event: plan
data: {"study_plan": [...]}

event: done
data: {}
```

## 3. 智能答疑

### POST /api/v1/chat/qa

基于课程知识库回答学生问题，支持苏格拉底式追问模式。

**请求体：**
```json
{
  "student_id": 1,
  "question": "什么是线性回归",
  "socratic": false
}
```

**参数说明：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| student_id | int | 是 | 学生 ID |
| question | str | 是 | 问题文本（最少 2 字） |
| socratic | bool | 否 | 是否开启苏格拉底追问模式，默认 false |

**响应：**
```json
{
  "answer": "线性回归是一种...（苏格拉底模式下末尾会追加引导性问题）",
  "source_refs": ["ml01_overview"],
  "agent_trace": {"agent_name": "答疑智能体(LLM)", "input_summary": "...", "output_summary": "..."}
}
```

## 4. 练习题自动批改

### POST /api/v1/quiz/grade

LLM 自动批改单道题目，支持选择题和简答题。

**请求体：**
```json
{
  "student_id": 1,
  "question": "什么是过拟合？",
  "student_answer": "模型在训练集上表现好但测试集差",
  "reference_answer": "过拟合是指模型在训练数据上表现优异但在新数据上泛化能力差...",
  "knowledge_unit": "模型评估"
}
```

**响应：**
```json
{
  "score": 85,
  "feedback": "回答基本正确，抓住了核心概念，但缺少对正则化缓解方法的描述。",
  "key_points_hit": ["训练集表现好", "测试集表现差"],
  "key_points_missed": ["正则化可以缓解过拟合", "交叉验证用于检测过拟合"],
  "knowledge_unit": "模型评估",
  "trend": "首次"
}
```

**评分规则：**
- 选择题：前端直接比较正确选项，不调用此接口
- 简答题：LLM 语义评分 0-100 分，返回命中/遗漏要点
- LLM 调用失败时 fallback 到关键词匹配评分

## 5. 评估与反馈

### POST /api/v1/assessment/submit

提交练习成绩，获取反馈和趋势分析。练习题页面答题后会自动调用此接口。

**请求体：**
```json
{"student_id": 1, "knowledge_unit": "线性回归", "score": 75}
```

**响应：**
```json
{
  "knowledge_unit": "线性回归",
  "score": 75,
  "feedback": "基本掌握，建议再做一轮巩固练习。",
  "next_recommendation": "继续练习 线性回归 的中等难度题",
  "trend": "上升"
}
```

### GET /api/v1/student/{student_id}/assessments

查询评估历史和弱项分析。

**响应：**
```json
{
  "records": [{"knowledge_unit": "...", "score": 75, "feedback": "...", "created_at": "..."}],
  "weak_points": [{"knowledge_unit": "...", "avg_score": 62.5, "attempts": 3}],
  "trend": {"线性回归": "上升", "模型评估": "下降"}
}
```

## 6. 学习看板

### GET /api/v1/student/{student_id}/dashboard

返回学生的完整学习状态：画像、诊断、资源、计划、评估记录、轨迹。用于前端刷新页面后恢复数据。

## 7. 知识库浏览

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/v1/kb/graph` | GET | 知识依赖图谱 JSON（节点 + 边），用于 ECharts 渲染 |
| `/api/v1/kb/modules` | GET | 课程模块索引（8 个模块） |
| `/api/v1/kb/questions` | GET | 题库（32 道题） |
| `/api/v1/kb/coding-cases` | GET | 代码案例（10 个案例） |

### GET /api/v1/kb/graph 响应示例

```json
{
  "nodes": [
    {
      "id": "ml01_overview",
      "label": "机器学习概述",
      "level": 1,
      "description": "机器学习的基本概念、任务分类、数据集划分",
      "knowledge_units": ["机器学习定义", "监督学习与无监督学习", "训练集与测试集"]
    }
  ],
  "edges": [
    {"from": "ml01_overview", "to": "ml03_linear_regression"},
    {"from": "ml03_linear_regression", "to": "ml04_logistic_regression"}
  ]
}
```
