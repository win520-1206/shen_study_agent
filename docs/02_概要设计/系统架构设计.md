# 系统架构设计

## 总体架构

- 前端：Vue 3 + Vite + Element Plus
- 后端：FastAPI
- 数据库：SQLite
- 知识库：Markdown + JSON 元数据
- 编排方式：显式多智能体角色协同

## 智能体角色

1. 画像智能体：抽取用户画像维度
2. 诊断智能体：识别知识短板与学习阶段
3. 资源规划智能体：决定需要生成的资源集合
4. 内容生成智能体：输出讲义、题目、代码案例等
5. 路径规划智能体：安排阶段化学习路线
6. 审查智能体：做基本事实性与安全性检查

## 数据流

1. 用户输入学习诉求
2. 画像智能体生成画像
3. 检索知识库命中模块
4. 诊断智能体输出优先学习点
5. 资源规划与内容生成完成个性化资料
6. 路径规划生成学习步骤
7. 审查智能体校验并输出结果
