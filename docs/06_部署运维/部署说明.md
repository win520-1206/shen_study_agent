# 部署说明

> 更新日期：2026-06-04

---

## 环境要求

- Python 3.13
- Node.js 18+
- npm（随 Node.js 安装）

## 本地开发

### 后端

```bash
pip install -r requirements.txt
uvicorn src.backend.app.main:app --reload
```

### 前端

```bash
cd src/frontend
npm install
npm run dev
```

## 生产部署

### 1. 服务器环境

```bash
# Python 3.13（通过 deadsnakes PPA）
apt install -y python3.13 python3.13-venv python3.13-dev

# Node.js 18（二进制安装，不要用 apt）
cd /tmp
curl -fsSL https://nodejs.org/dist/v18.20.4/node-v18.20.4-linux-x64.tar.xz -o node18.tar.xz
tar -xf node18.tar.xz
cp -f node-v18.20.4-linux-x64/bin/node /usr/local/bin/node
cp -rf node-v18.20.4-linux-x64/lib/node_modules /usr/local/lib/node_modules
ln -sf /usr/local/lib/node_modules/npm/bin/npm-cli.js /usr/local/bin/npm
```

### 2. 项目部署

```bash
cd /opt/shen_study_agent
git clone https://github.com/win520-1206/shen_study_agent.git .

# 创建虚拟环境
python3.13 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 配置
cp src/backend/app/config.example.py src/backend/app/config.py
# 编辑 config.py 填入 API Key

# 构建前端
cd src/frontend
npm install
node node_modules/vite/bin/vite.js build
cd /opt/shen_study_agent
```

### 3. 配置 systemd

```bash
cat > /etc/systemd/system/learnmate.service << 'EOF'
[Unit]
Description=LearnMate-AI Backend Service
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/shen_study_agent
ExecStart=/opt/shen_study_agent/venv/bin/python -m uvicorn src.backend.app.main:app --host 0.0.0.0 --port 8000
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
EOF

systemctl daemon-reload
systemctl enable learnmate
systemctl start learnmate
```
