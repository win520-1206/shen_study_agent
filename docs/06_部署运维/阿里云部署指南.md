# 阿里云部署指南

> 更新日期：2026-05-30
> 适用系统：Ubuntu 22.04 LTS

---

## 一、购买阿里云服务器

### 1.1 推荐配置

| 项目 | 推荐值 |
|------|--------|
| 实例类型 | ECS 经济型 (ecs.t6-c1m1.large) |
| CPU/内存 | 1核 2G |
| 操作系统 | Ubuntu 22.04 LTS 64位 |
| 存储 | 40G 高效云盘 |
| 带宽 | 1-3 Mbps（按量付费） |
| 地域 | 华东1-杭州（或离你最近的） |

### 1.2 购买流程

1. 访问 [阿里云官网](https://www.aliyun.com)
2. 注册账号并完成实名认证
3. 搜索"云服务器 ECS"
4. 点击"立即购买"
5. 选择配置：
   - 地域：华东1-杭州
   - 实例：ecs.t6-c1m1.large
   - 镜像：Ubuntu 22.04 LTS 64位
   - 存储：40G 高效云盘
   - 网络：默认 VPC
   - 安全组：创建新安全组，开放 8000 端口
   - 设置 root 密码
6. 确认购买

### 1.3 费用参考

- 新用户优惠：约 9-30 元/月
- 学生认证：可能有更低价格（建议查看学生优惠页面）

## 二、配置安全组

### 2.1 进入安全组配置

1. 登录阿里云控制台
2. 进入 ECS 实例详情页
3. 点击左侧菜单"安全组"
4. 点击"配置规则"

### 2.2 添加入站规则

点击"手动添加"，填写：

| 授权策略 | 优先级 | 协议类型 | 端口范围 | 授权对象 |
|----------|--------|----------|----------|----------|
| 允许 | 1 | 自定义 TCP | 8000/8000 | 0.0.0.0/0 |

## 四、部署项目

### 4.1 上传项目代码

**方式一：SCP 命令（最快）**

```bash
# 本地 PowerShell 运行
scp -r C:\shen_study_agent root@你的IP:/opt/learnmate-ai
```

**方式二：Git（推荐，方便后续更新）**

1. 在 GitHub 创建仓库（不要上传 config.py！）
2. 本地：
   ```bash
   cd C:\shen_study_agent
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/learnmate-ai.git
   git push -u origin main
   ```
3. 服务器：
   ```bash
   ssh root@你的IP
   cd /opt
   git clone https://github.com/你的用户名/learnmate-ai.git
   ```

### 4.2 在服务器上创建配置文件

**重要：config.py 不能上传到 GitHub（包含 API 密钥）**

```bash
# 登录服务器
ssh root@你的IP

# 进入项目目录
cd /opt/learnmate-ai

# 创建 config.py（用示例文件复制）
cp src/backend/app/config.example.py src/backend/app/config.py

# 编辑 config.py，填入你的 API 密钥
nano src/backend/app/config.py
```

在 nano 编辑器中：
- 找到 `LLM_API_KEY_V2 = ""`
- 改为 `LLM_API_KEY_V2 = "sk-你的密钥"`
- 按 Ctrl+O 保存，按 Ctrl+X 退出

### 4.3 运行部署脚本

```bash
# 添加执行权限
chmod +x deploy.sh

# 运行部署脚本
sudo ./deploy.sh
```

脚本会自动完成：
1. 更新系统
2. 安装 Python 和 Node.js
3. 安装依赖
4. 构建前端
5. 配置 systemd 服务
6. 启动服务

### 4.4 手动部署（如果脚本失败）

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装依赖
sudo apt install -y python3 python3-pip python3-venv git

# 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo bash -
sudo apt install -y nodejs

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装 Python 依赖
pip install -r requirements.txt

# 构建前端
cd src/frontend
npm install
npm run build
cd ../..

# 启动服务
nohup python3 -m uvicorn src.backend.app.main:app --host 0.0.0.0 --port 8000 &
```




```bash
cd /opt/learnmate-ai
git pull

```

## 七、常见问题

### 7.1 无法访问页面

1. 检查安全组是否开放 8000 端口
2. 检查服务器防火墙：
   ```bash
   sudo ufw status
   sudo ufw allow 8000
   ```
3. 检查服务是否运行：
   ```bash
   systemctl status learnmate
   ```

### 7.2 服务启动失败

查看日志：
```bash
journalctl -u learnmate -n 100
```

### 7.3 前端页面空白

重新构建前端：
```bash
cd /opt/learnmate-ai/src/frontend
npm run build
systemctl restart learnmate
```

### 7.4 Python 依赖安装失败

确保使用虚拟环境：
```bash
cd /opt/learnmate-ai
source venv/bin/activate
pip install -r requirements.txt
```

## 八、安全建议

1. **修改 SSH 端口**（可选）：减少暴力破解
2. **使用密钥登录**：比密码更安全
3. **定期更新系统**：`sudo apt update && sudo apt upgrade`
4. **备份数据库**：定期备份 `/opt/learnmate-ai/runtime/learnmate.db`

## 九、成本优化

- **按量付费**：适合测试和低流量场景
- **包年包月**：适合长期使用，价格更低
- **学生优惠**：查看阿里云学生特惠页面
- **竞价实例**：价格最低，但可能被回收（不推荐生产环境）
