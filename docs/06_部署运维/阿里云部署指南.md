# 阿里云部署指南

> 更新日期：2026-05-30  
> 适用系统：Ubuntu 22.04 LTS  
> 推荐 Python：**3.13**

---

## 一、购买阿里云服务器

### 1.1 推荐配置

| 项目 | 推荐值 |
|------|--------|
| 实例类型 | ECS 经济型 (ecs.t6-c1m1.large) |
| CPU/内存 | 1核 2G |
| 操作系统 | Ubuntu 22.04 LTS 64位 |
| 存储 | 40G 高效云盘 |
| 带宽 | 1-3 Mbps（按量付费） |
| 地域 | 华东1-杭州（或离你最近的） |

### 1.2 购买流程

1. 访问 [阿里云官网](https://www.aliyun.com)
2. 注册账号并完成实名认证
3. 搜索"云服务器 ECS"
4. 点击"立即购买"
5. 选择配置：
   - 地域：华东1-杭州
   - 实例：ecs.t6-c1m1.large
   - 镜像：Ubuntu 22.04 LTS 64位
   - 存储：40G 高效云盘
   - 网络：默认 VPC
   - 安全组：创建新安全组，开放 8000 端口
   - 设置 root 密码
6. 确认购买

## 五、服务器上创建配置文件

```bash
ssh root@你的IP
cd /opt/shen_study_agent
cp src/backend/app/config.example.py src/backend/app/config.py
nano src/backend/app/config.py
```

把下面几项改成你的真实配置：

```python
USE_LLM = True

LLM_BASE_URL = "https://api.deepseek.com"
```

- 回车
- `Ctrl + X`

## 六、快速部署（推荐）

```bash
cd /opt/shen_study_agent
chmod +x deploy.sh
sudo ./deploy.sh
```

脚本会自动：
1. 安装 Python 3.13
2. 创建 Python 3.13 虚拟环境
3. 安装依赖
4. 构建前端
5. 配置 `systemd`
6. 启动服务

## 七、手动部署（如果脚本失败）

### 7.1 安装 Python 3.13

```bash
apt update
apt install -y software-properties-common
add-apt-repository ppa:deadsnakes/ppa -y
apt update
apt install -y python3.13 python3.13-venv python3.13-dev python3-pip git curl
```

验证：
```bash
python3.13 --version
```

### 7.2 重建虚拟环境

```bash
cd /opt/shen_study_agent
rm -rf venv
python3.13 -m venv venv
source venv/bin/activate
python --version
```

### 7.3 安装依赖

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 7.4 构建前端

```bash
apt install -y nodejs npm
cd src/frontend
npm install
npm run build
cd ../..
```

### 7.5 启动服务测试

```bash
python -m uvicorn src.backend.app.main:app --host 0.0.0.0 --port 8000
```

看到类似下面内容说明成功：

```bash
Uvicorn running on http://0.0.0.0:8000
```

然后本机浏览器访问：

```text
http://你的服务器公网IP:8000
http://你的服务器公网IP:8000/docs
```

## 八、配置 systemd 开机自启

创建服务文件：

```bash
cat > /etc/systemd/system/learnmate.service << EOF
[Unit]
Description=LearnMate-AI Backend Service
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/shen_study_agent
Environment="PATH=/opt/shen_study_agent/venv/bin"
ExecStart=/opt/shen_study_agent/venv/bin/python -m uvicorn src.backend.app.main:app --host 0.0.0.0 --port 8000
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
EOF
```

启用并启动：

```bash
systemctl daemon-reload
systemctl enable learnmate
systemctl restart learnmate
```

## 十、常见问题

### 10.1 浏览器打不开

先检查：

```bash
ss -ltnp | grep 8000
curl http://127.0.0.1:8000
curl http://你的服务器公网IP:8000
```

判断：
- `127.0.0.1` 通但公网不通：优先查安全组 / 防火墙
- 两个都不通：应用没启动成功

### 10.2 `ImportError: cannot import name 'UTC'`

说明你还在用 Python 3.10。  
按上面的步骤切到 Python 3.13 并重建虚拟环境即可。

### 10.3 页面空白

重新构建前端：

```bash
cd /opt/shen_study_agent/src/frontend
npm run build
systemctl restart learnmate
```
