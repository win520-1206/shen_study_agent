# LearnMate-AI 部署成果简版

> 用途：适合比赛汇报、项目简介、答辩讲解时快速说明部署成果。
> 更新日期：2026-06-04

---

## 1. 部署目标

将 LearnMate-AI 从本地开发原型升级为可公网访问的在线系统，使评委、老师和同学无需配置本地环境即可直接体验系统全部功能。

---

## 2. 已完成的部署成果

1. 后端服务（FastAPI + 14 个 API 端点）在云端稳定运行。
2. 前端页面（Vue 3 + vue-router + ECharts 多页面 SPA）已完成构建并由后端统一提供访问。
3. 5 个 LLM 智能体已接入 DeepSeek API，支持 Mock/LLM 双模式切换。
4. 知识图谱模块已部署，支持 ECharts 可视化。
5. 交互式答题与自动批改功能已上线。
6. 学生持续跟踪功能已实现（localStorage 持久化）。
7. 系统可通过公网 IP 直接访问。
8. 接口文档可在线查看。
9. 服务已接入 `systemd`，支持后台运行、重启与开机自启。

## 4. 系统能力

部署完成后，系统已具备以下完整能力：

1. 结构化表单 + 3 种预设快速创建学生画像
2. 6+1 个智能体协作生成个性化学习方案
3. 5 类个性化学习资源生成（讲义、练习题、代码案例、思维导图、学习路径）
4. 知识图谱可视化与薄弱点高亮诊断
5. 交互式练习题（选择题即时判对错 + 简答题 LLM 自动批改）
6. 苏格拉底式智能答疑（主动追问引导思考）
7. 学生持续跟踪（刷新不丢失，多学生切换）
8. 评估历史与学习趋势分析
9. 教师/评委成果看板
10. SSE 流式输出逐个渲染资源

---

## 5. 技术亮点

1. **前后端一体化部署**：前端构建产物由后端统一托管，访问入口统一。
2. **Python 3.13 + Node.js 18**：使用最新稳定版本，二进制安装避免系统包管理器冲突。
3. **systemd 管理**：支持后台运行、重启、开机自启，稳定性高。
4. **Git 强制同步**：使用 `git fetch + git reset --hard` 避免 merge 冲突，更新可靠。
5. **一行配置切换 LLM**：`config.py` 中 `USE_LLM` 开关控制 Mock/LLM 模式。
6. **多智能体可解释性**：每个智能体的输入输出摘要、决策原因均可追溯。
