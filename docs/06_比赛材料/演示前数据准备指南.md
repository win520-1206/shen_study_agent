# 比赛演示前数据准备指南

> 用途：比赛演示前清空测试数据，跑几次典型预设，让系统展示效果最干净、最有说服力。
> 更新日期：2026-06-04

---

## 第 1 步：备份当前数据库

在服务器上执行：

```bash
cp /opt/shen_study_agent/runtime/learnmate.db /opt/shen_study_agent/runtime/learnmate.db.bak
```

---

## 第 2 步：清空数据库

```bash
rm -f /opt/shen_study_agent/runtime/learnmate.db
systemctl restart learnmate
```

重启后系统会自动创建一个新的空数据库。

## 第 3 步：跑 3 个典型预设

### 零基础入门型

```bash
# 创建学生
STUDENT_ID=$(curl -s -X POST http://127.0.0.1:8000/api/v1/students \
  -H "Content-Type: application/json" \
  -d '{"name":"李同学","major":"人工智能","target_course":"机器学习基础"}' \
  | python3.13 -c "import sys,json; print(json.load(sys.stdin)['id'])")

echo "零基础学生 ID: $STUDENT_ID"

# 构建画像 + 生成资源
curl -s -X POST http://127.0.0.1:8000/api/v1/chat/profile-build \
  -H "Content-Type: application/json" \
  -d "{\"student_id\":$STUDENT_ID,\"message\":\"我是人工智能专业学生，机器学习零基础，希望通俗讲解，先建立整体认知，不用一开始就上很多代码。练习偏好混合题型。\"}" \
  | python3.13 -c "import sys,json; d=json.load(sys.stdin); print('画像:', d['student']['profile']['learning_goal']); print('资源数:', len(d['resources'])); print('推荐依据:', d['recommendation_summary'][:80])"

# 提交评估（模拟低分）
curl -s -X POST http://127.0.0.1:8000/api/v1/assessment/submit \
  -H "Content-Type: application/json" \
  -d "{\"student_id\":$STUDENT_ID,\"knowledge_unit\":\"线性回归\",\"score\":55}"
```
