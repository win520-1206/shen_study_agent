## 1. 项目一句话介绍

> **LearnMate-AI 是一个基于大模型的个性化学习多智能体系统。学生只需通过结构化表单描述自己的学习情况，7 个智能体就会自动协作，为他生成一套只适合他的学习方案——包括讲义、练习题、代码案例、思维导图和学习路径，并支持知识图谱诊断、交互式答题自动批改和学生持续跟踪。**

### 30 秒版本

> 传统课堂"一套教案教所有人"，LearnMate-AI 让每个学生都有专属学习方案。系统通过结构化表单快速了解学生的基础、目标和薄弱环节，由 7 个智能体分工协作生成个性化学习资源。更有知识图谱诊断薄弱点依赖关系、交互式答题即时批改、苏格拉底式追问引导思考，实现真正的因材施教。

### 1 分钟版本

> 大学课程学习中，每个学生的基础、目标和学习方式都不同，但传统教学很难做到真正的"因材施教"。LearnMate-AI 通过大模型和多智能体技术解决了这个问题。学生通过结构化表单选择学习目标、基础水平、薄弱知识点等 8 个维度，系统自动构建学习画像，识别知识短板，然后由 7 个智能体协作生成个性化学习资源。与普通 AI 问答不同，LearnMate-AI 有三大杀手锏：第一，知识图谱诊断——基于学科依赖关系定位薄弱点，建议先巩固前置知识；第二，交互式答题——选择题即时判对错，简答题 AI 自动评分并给出要点反馈；第三，苏格拉底式追问——AI 不只回答，还会主动引导学生深入思考。零基础学生和有经验的学生会得到完全不同的学习方案。

## 2. 项目核心问题

### 要解决的问题

**"同一个课堂里，每个学生的学习起点、目标和方式都不一样，但现有的学习工具很难做到真正的个性化。"**

### 问题拆解

| 问题 | 具体表现 |
|------|----------|
| 资源不成体系 | 讲义、习题、代码案例分散在不同平台 |
| 不知道学什么 | 学生不清楚薄弱环节，盲目学习效率低 |
| 个性化不足 | 大多数 AI 工具只做"问答"，不会根据学生调整内容 |
| 缺乏路径感 | 学完一个知识点不知道下一步该学什么 |
| 内容不可靠 | 纯大模型生成容易出现幻觉 |
| 练习无反馈 | 做完题没有即时批改，不知道对错原因 |
| 学习不持续 | 每次使用都是独立的，无法跨次跟踪 |

## 3. 系统能力总览

```
学生通过结构化表单描述学习情况
      ↓
┌─────────────────────────────────────────────┐
│  画像智能体：构建 8 维学习画像               │
│  ↓                                          │
│  诊断智能体：分析薄弱环节 + 知识图谱定位     │
│  ↓                                          │
│  资源规划智能体：决定资源难度和风格          │
│  ↓                                          │
│  内容生成智能体：生成 5 类个性化资源         │
│  ↓                                          │
│  路径规划智能体：设计分阶段学习路径          │
│  ↓                                          │
│  审查智能体：校验内容完整性                 │
│  ↓                                          │
│  自动批改智能体：交互式答题评分 + 反馈       │
└─────────────────────────────────────────────┘
      ↓
输出：画像 + 诊断 + 知识图谱 + 5 类资源 + 学习路径 + 练习批改
```

### 7 个智能体的分工

| 智能体 | 职责 | 类比 |
|--------|------|------|
| 画像智能体 | 从表单信息提取 8 维学习画像 | 学情调查员 |
| 诊断智能体 | 分析薄弱环节 + 知识图谱依赖分析 | 学情分析师 |
| 资源规划智能体 | 根据画像决定资源参数 | 课程顾问 |
| 内容生成智能体 | 生成讲义、题目、案例等 | 备课老师 |
| 路径规划智能体 | 设计分阶段学习任务 | 学习教练 |
| 审查智能体 | 校验内容完整性和来源 | 质检员 |
| 自动批改智能体 | 练习题 LLM 评分和反馈 | AI 助教 |

## 4. 三大创新亮点

### 创新点 1：知识图谱盲区诊断

- 内置课程知识依赖图谱（8 个模块、依赖关系边）
- 前端 ECharts 可视化渲染
- 薄弱知识点自动高亮 + 上游依赖提示
- 将"个性化"从推资源提升到**学科逻辑推演**层面
- 例如：发现"逻辑回归"薄弱 → 提示"建议先巩固梯度下降基础"

### 创新点 2：交互式答题与自动批改

- 选择题（A/B/C/D）：点击选项**即时判对错**，前端直接比较
- 简答题：LLM 自动评分 0-100 分，返回**命中要点 + 遗漏要点**
- 答题后**自动提交评估成绩**，无需手动操作
- 成绩累积影响后续推荐策略

### 创新点 3：苏格拉底式追问

- AI 不仅回答问题，还**主动引导学生深入思考**
- 回答末尾追加引导性问题
- 例如：学生问"什么是过拟合？"，AI 回答后追问"你能想出一种正则化方法来缓解它吗？"
- 体现 AI 作为"助教"的**主动引导能力**

### 额外创新点

- **学生持续跟踪**：localStorage 持久化，刷新不丢失，多学生切换
- **结构化表单**：8 个维度精确画像，比自然语言输入更稳定
- **多智能体可解释性**：每个智能体的决策过程可追溯

## 6. 技术栈

- 后端：FastAPI + SQLAlchemy + SQLite
- 前端：Vue 3 + Vite + Element Plus + vue-router + ECharts
- 大模型：DeepSeek API（支持一行配置切换）
- 知识库：8 个 ML 模块（讲义 + 32 道题 + 10 个代码案例 + 知识图谱）
- 部署：阿里云 ECS + systemd
