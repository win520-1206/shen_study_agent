# 数据库设计

> 更新日期：2026-06-04
> 数据库：SQLite（`runtime/learnmate.db`）

---

## students

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| name | TEXT | 学生姓名 |
| major | TEXT | 专业 |
| target_course | TEXT | 目标课程 |
| profile_json | TEXT | 8 维画像 JSON |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

## learning_sessions

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| student_id | INTEGER FK | 关联学生 |
| user_input | TEXT | 用户输入的原始诉求 |
| diagnosis_json | TEXT | 诊断结果 JSON |
| plan_json | TEXT | 学习计划 JSON |
| created_at | DATETIME | 创建时间 |

## generated_resources

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| session_id | INTEGER FK | 关联学习会话 |
| resource_type | TEXT | 资源类型（lesson_note/quiz/coding_case/mind_map/study_path） |
| title | TEXT | 资源标题 |
| content | TEXT | 资源内容（Markdown） |
| source_refs | TEXT | 来源引用 JSON 列表 |
| created_at | DATETIME | 创建时间 |

## assessment_records

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| student_id | INTEGER FK | 关联学生 |
| knowledge_unit | TEXT | 知识单元名称 |
| score | INTEGER | 分数（0-100） |
| feedback | TEXT | 反馈文字 |
| created_at | DATETIME | 创建时间 |

## agent_traces

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| session_id | INTEGER FK | 关联学习会话 |
| agent_name | TEXT | 智能体名称 |
| input_summary | TEXT | 输入摘要 |
| output_summary | TEXT | 输出摘要 |
| created_at | DATETIME | 创建时间 |

---

## 关系图

```
students 1──N learning_sessions
students 1──N assessment_records
learning_sessions 1──N generated_resources
```
