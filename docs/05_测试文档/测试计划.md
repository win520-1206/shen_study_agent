# 测试计划

> 更新日期：2026-06-04
> 版本：V1.0

---

## 目标

验证系统满足赛题核心要求 + 全部加分项功能正常。

## 关键场景

### 核心功能（必须通过）

1. 新学生建档成功（POST /students）
2. 结构化表单信息能被正确拼接为学习诉求
3. 对话能构建 8 维画像
4. 系统能返回 5 类资源
5. 系统能返回 3 阶段学习路径
6. 不同学生输入产生不同结果（个性化差异）
7. 学习记录能进入 Dashboard
8. 评估成绩自动提交并触发后续推荐差异

### LLM 模式

9. USE_LLM=True 时 5 个 LLM 智能体正常工作
10. USE_LLM=False 时规则式智能体正常工作
11. LLM 画像能正确处理否定句和复杂意图
12. LLM 内容生成质量优于模板

### 智能答疑 + 苏格拉底追问（加分项）

13. POST /api/v1/chat/qa（socratic=false）能回答课程问题
14. 回答基于知识库内容，有 source_refs 标注
15. POST /api/v1/chat/qa（socratic=true）回答末尾追加引导性问题
16. 引导性问题与当前话题相关

### 知识图谱（加分项）

17. GET /api/v1/kb/graph 返回节点和边
18. 前端 ECharts 正确渲染图谱
19. 薄弱知识点及上游依赖正确高亮
20. 点击节点显示模块描述

### 交互式练习题与自动批改（加分项）

21. 练习题正确解析为选择题 + 简答题混合格式
22. 选择题 A/B/C/D 选项正确渲染
23. 选择题点击选项即时判对错
24. 简答题 LLM 批改返回分数、反馈、要点
25. 答题后成绩自动提交到评估记录
26. exercise_preference 影响题型分配

### 学生持续跟踪

27. localStorage 正确保存学生 ID 和姓名
28. 刷新页面后数据自动恢复
29. 侧边栏学生选择器正确显示学生列表
30. 切换学生后数据正确切换
31. 同一学生多次生成不重复创建记录

### 评估与闭环

32. 评估历史记录完整
33. 弱项识别准确（avg < 70 标记为弱项）
34. 趋势分析正确（上升/下降/首次）
35. 二次对话时弱项被优先推荐

### 流式输出

36. SSE 流式端点正常返回事件
37. 前端能逐个渲染资源卡片

### 多页面路由

38. vue-router 6 条路由全部可访问
39. 侧边栏导航项高亮正确
40. 未生成时资源导航项灰色禁用

### 异常处理

41. 空输入时系统不崩溃
42. 超长输入时系统不崩溃
43. 不存在的学生 ID 返回 404
44. LLM 调用失败时 fallback 到 Mock
