# 测试计划

> 更新日期：2026-05-30
> 版本：V0.4

---

## 目标

验证系统满足赛题核心要求 + 加分项功能正常。

## 关键场景

### 核心功能（必须通过）

1. 新学生建档成功
2. 对话能构建 8 维画像
3. 系统能返回 5 类资源
4. 系统能返回 3 阶段学习路径
5. 不同学生输入产生不同结果（个性化差异）
6. 学习记录能进入 Dashboard
7. 测验成绩能触发后续推荐差异

### LLM 模式（新增）

8. USE_LLM=True 时 LLM 智能体替代规则式智能体
9. USE_LLM=False 时规则式智能体正常工作
10. LLM 画像能正确处理否定句
11. LLM 诊断推荐策略更自然
12. LLM 内容生成质量优于模板

### 智能答疑（加分项）

13. POST /api/v1/chat/qa 能回答课程问题
14. 回答基于知识库内容，有 source_refs 标注
15. 能处理不同类型的问题（概念、对比、原理）

### 评估增强（加分项）

16. 评估历史记录完整
17. 弱项识别准确（avg < 70 标记为弱项）
18. 趋势分析正确（上升/下降/持平）
19. 二次对话时弱项被优先推荐

### 流式输出（加分项）

20. SSE 流式端点正常返回事件
21. 前端能逐个渲染资源卡片
22. 非流式模式不受影响

### 异常处理

23. 空输入时系统不崩溃
24. 超长输入时系统不崩溃
25. 不存在的学生 ID 返回 404
26. API 调用失败时 fallback 到 Mock

## 手工验证建议

1. 启动后端：`uvicorn src.backend.app.main:app --reload`
2. 启动前端：`cd src/frontend && npx vite dev`
3. 选择 3 种学生预设，对比结果差异
4. 勾选"流式输出模式"，观察资源逐个出现
5. 在智能答疑区输入问题，检查回答质量
6. 提交不同分数的评估，检查趋势和弱项
