# 用户使用说明书

> 更新日期：2026-05-30
> 版本：V0.4

---

## 1. 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+

### 启动后端

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
uvicorn src.backend.app.main:app --reload
```

后端运行在 `http://127.0.0.1:8000`，API 文档：`http://127.0.0.1:8000/docs`

### 启动前端

```bash
cd src/frontend
npm install
npm run dev
```

前端运行在 `http://localhost:5173`

---

## 2. 使用演示

### 方式一：一键预设

1. 打开前端页面
2. 点击顶部的预设按钮（零基础入门型 / 项目实战型 / 考试冲刺型）
3. 点击"生成个性化学习方案"
4. 等待 6 个智能体协作完成
5. 查看结果：画像、诊断、管线、资源、路径

### 方式二：自定义输入

1. 在"姓名"和"专业"字段输入信息
2. 在"学习诉求"文本框描述你的学习情况
3. 点击"生成个性化学习方案"

**示例输入：**
- "我是人工智能专业学生，机器学习零基础，希望通俗讲解，先建立整体认知。"
- "我有 Python 基础，想通过项目实战学会特征工程和模型评估。"
- "期末考试临近，我想快速掌握高频考点和易错题型。"

### 方式三：智能答疑

1. 先完成一次"生成个性化学习方案"
2. 在页面底部的"智能答疑"区域输入问题
3. 点击"发送"或按回车

**示例问题：**
- "什么是线性回归？"
- "逻辑回归和线性回归有什么区别？"
- "什么是过拟合？"

## 3. 查看学习效果评估

1. 提交练习成绩：通过 API `POST /api/v1/assessment/submit`
2. 查看评估历史：通过 API `GET /api/v1/student/{id}/assessments`
3. 前端会自动展示评估历史和弱项分析

---

## 4. 切换大模型

在 `src/backend/app/config.py` 中修改配置：

```python
USE_LLM = True
LLM_PROVIDER = "generic"
LLM_BASE_URL = "https://api.deepseek.com"  # 你的 API 地址
LLM_API_KEY_V2 = "your-api-key"             # 你的 API 密钥
LLM_MODEL = "deepseek-chat"                  # 模型名称
```

重启后端即可生效。
