# LearnMate-AI 项目说明

LearnMate-AI 是一个面向 **2026 年中国软件杯 A3 赛题** 的训练式项目实现。  
赛题名称为：**基于大模型的个性化资源生成与学习多智能体系统开发**。

这个仓库同时服务两个目标：

1. 做出一个可运行、可演示、可继续扩展的比赛 MVP。
2. 从项目一开始就保留好文档、代码和证据材料，满足软著申请与比赛答辩需要。

## 一、项目定位

本项目当前聚焦于 **“机器学习基础”** 课程场景，目标是构建一个能帮助学生进行个性化学习的多智能体系统。

系统的核心能力包括：

- 通过自然语言对话构建学生画像
- 基于学生画像识别学习短板
- 通过多智能体协同生成个性化学习资源
- 自动规划学习路径
- 根据学习结果给出后续推荐
- 保留智能体轨迹，方便比赛演示和系统解释

## 二、当前 MVP 范围

当前版本已经覆盖以下核心能力：

- 8 维学生画像构建
- 6 个智能体的显式协同设计
- 至少 5 类学习资源生成
- 学习路径规划与推荐
- 基础测验反馈闭环
- 可用于演示的智能体轨迹记录

当前资源类型包括：

1. 知识讲解文档
2. 练习题
3. 代码实操案例
4. 思维导图文本
5. 学习任务清单

## 三、项目目录结构

- `src/backend`：FastAPI 后端，负责接口、数据库、多智能体编排
- `src/frontend`：Vue 3 前端演示页
- `data/course_kb`：机器学习基础课程知识库
- `docs`：需求、设计、测试、部署、软著相关文档
- `evidence`：截图、版本日志、测试记录等留痕材料
- `tests`：后端自动化测试

## 四、快速启动

### 方式一：一键启动（推荐）

双击 `start.bat`，浏览器访问 `http://localhost:8000` 即可。

局域网内其他人访问：`http://你的IP:8000`

### 方式二：公网分享（手机也能用）

1. 安装 [cloudflared](https://developers.cloudflare.com/cloudflare-one/connections/connect-apps/install-and-setup/)
2. 双击 `start_public.bat`
3. 终端会显示一个公网 URL（类似 `https://xxx.trycloudflare.com`）
4. 把 URL 发给任何人，手机/电脑浏览器打开就能用

### 方式三：开发模式

```bash
# 后端
pip install -r requirements.txt
uvicorn src.backend.app.main:app --reload

# 前端（另一个终端）
cd src/frontend
npm install
npm run dev
```

启动后可访问：
- 主页面：`http://localhost:8000`
- API 文档：`http://localhost:8000/docs`

## 五、推荐学习顺序

如果你是边做边学，建议按下面顺序推进：

1. 阅读 `docs/01_立项与需求/项目简介.md`
2. 阅读 `docs/01_立项与需求/赛题需求清单.md`
3. 阅读 `docs/01_立项与需求/第1周训练任务.md`
4. 查看 `data/course_kb` 中的课程知识库
5. 启动后端，测试 `/api/v1/chat/profile-build`
6. 查看前端页面，理解完整演示流程
7. 每天把进展记录到 `evidence/version_records`
