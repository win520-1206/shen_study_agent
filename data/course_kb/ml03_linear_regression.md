# 线性回归

## 学习目标

- 理解线性回归适合解决什么类型的问题
- 掌握“输入特征如何线性影响输出”的基本思想
- 理解损失函数、参数拟合和评价指标之间的关系
- 能够用 scikit-learn 跑通一个基础回归实验，并解释主要步骤

## 模块导入

线性回归通常是很多人接触的第一个机器学习模型。  
它之所以重要，不只是因为简单，而是因为它能帮助你建立一个非常关键的认知：

**模型训练其实就是在寻找一组参数，使预测结果尽量接近真实结果。**

学懂线性回归后，你会更容易理解逻辑回归、神经网络和其他优化型模型。

## 核心知识点

### 1. 线性回归解决什么问题

线性回归主要用于预测连续值，例如：

- 房价预测
- 销量预测
- 温度预测
- 成绩预测

它假设输出可以由输入特征的线性组合来近似表示。

### 2. 模型形式的直观理解

线性回归的核心思想是：

输出 = 截距 + 各个特征乘以对应权重后再求和

这里的权重表示“某个特征变化时，对结果会产生多大影响”。  
虽然真实世界不一定完全线性，但线性回归提供了一个很好的起点。

### 3. 最小二乘法与损失函数

训练线性回归模型时，我们要让预测值和真实值尽量接近。  
常见做法是最小化误差平方和，也就是最小二乘思想。

为什么要平方？

- 避免正负误差相互抵消
- 让大误差受到更重惩罚
- 数学上更容易优化

### 4. 回归评价指标

常见指标包括：

- MSE：均方误差，关注平均平方误差大小
- RMSE：均方根误差，量纲与原始目标值一致，更容易直观理解
- R²：决定系数，表示模型解释数据波动的能力

这些指标没有绝对“最好”的一个，但要知道它们分别在强调什么。

### 5. 线性回归的优点与局限

优点：

- 简单易懂
- 训练快
- 可解释性强

局限：

- 对非线性关系表达能力有限
- 容易受异常值影响
- 需要注意特征之间的相关性问题

## 关键流程理解

一个线性回归实验通常包括：

1. 准备特征和目标值
2. 划分训练集和测试集
3. 创建模型并训练
4. 在测试集上预测
5. 使用 MSE、RMSE、R² 评价结果
6. 分析误差并继续优化

## 常见误区

- 把“线性回归”理解成“只能处理一维直线”
- 把相关性直接等同于因果关系
- 只看训练误差，不看测试集表现
- 看到 R² 高就认为模型一定有业务价值

## 实践建议

- 解释为什么线性回归适合房价预测这类任务
- 比较不同训练测试划分比例下的模型表现
- 尝试加入一个无关特征，观察模型效果变化
