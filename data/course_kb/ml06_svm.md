# 支持向量机

## 学习目标

- 理解支持向量机的最大间隔分类思想
- 认识支持向量、间隔和分类边界之间的关系
- 理解核函数为什么能处理非线性问题
- 知道 SVM 在实践中对参数和特征缩放较敏感

## 模块导入

支持向量机常被认为“概念稍难，但思想很漂亮”。  
它的核心不是简单地找一条分界线，而是尽量找到一条**离两类样本都更远**的分界线。

这个“尽量远”的想法，就是最大间隔思想。

## 核心知识点

### 1. 什么是分类边界

在分类问题中，我们往往希望找到一条边界，把不同类别样本分开。  
如果有很多条边界都能分开样本，SVM 更倾向于选择那条“安全距离更大”的边界。

### 2. 最大间隔思想

间隔可以理解为分类边界到两类最近样本的距离。  
SVM 的核心目标是让这个间隔尽可能大，因为更大的间隔通常意味着更好的泛化能力。

### 3. 支持向量

真正决定分类边界位置的，不是所有样本，而是离边界最近的一些关键样本，这些样本叫支持向量。  
这也是“支持向量机”名称的来源。

### 4. 软间隔与参数 C

现实数据经常不是完全可分的。  
因此 SVM 会在“间隔尽量大”和“分类错误尽量少”之间做权衡。

参数 C 的作用可以理解为：

- C 较大：更强调减少训练误差
- C 较小：更强调更宽的间隔和更强的泛化

### 5. 核函数与非线性分类

当数据在原空间中无法线性分开时，可以通过核函数把问题映射到更高维空间，使其更容易被分开。  
常见核函数包括线性核、多项式核、RBF 核等。

## 关键流程理解

使用 SVM 时通常需要注意：

1. 先做特征预处理，尤其是特征缩放
2. 选择核函数
3. 调整 C 等参数
4. 使用验证集或交叉验证比较不同参数效果

## 常见误区

- 不理解最大间隔思想，只记住“这是一个分类模型”
- 不做特征缩放就直接训练 SVM
- 不理解参数意义就机械调参
- 把核函数理解成“神奇黑盒”，而不是一种映射思想

## 实践建议

- 比较做缩放和不做缩放时 SVM 的效果差异
- 思考为什么只有部分样本会成为支持向量
- 用自己的话解释“线性不可分”是什么意思
