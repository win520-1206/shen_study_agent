## 今日遇到的问题

### 1. 多 Python 环境混用

当前机器上至少存在以下 Python 路径：

- `C:\\python\\python.exe`
- `C:\\Users\\23120\\anaconda3\\python.exe`

默认 `python` 与当前项目依赖环境不一致，容易导致：

- 找不到 `fastapi`
- 找不到 `pytest`
- 同一命令前后行为不一致

### 2. PowerShell 配置文件提示仍然存在

终端每次执行命令前仍会尝试加载 `profile.ps1`，造成脚本策略报错提示。  
目前确认这是环境噪音，不影响项目功能验证，但会影响命令输出整洁度。
