## 今日关键概念学习

### 组件化开发

将一个大文件拆分成多个小文件，每个文件负责一个独立的功能。好处：
- 代码可维护性：改一个功能不用翻整个文件
- 可复用性：组件可以在不同地方使用
- 团队协作：不同人可以同时开发不同组件

### Props 和 Emits

Vue 3 组件间通信的两种方式：
- Props：父组件向子组件传递数据（单向数据流）
- Emits：子组件向父组件发送事件（向上通知）

### CSS 变量（Design Tokens）

用 `--变量名: 值` 定义可复用的设计常量，如颜色、间距、字体。修改一处，全局生效。

### 玻璃质感（Glassmorphism）

通过 `backdrop-filter: blur()` + 半透明背景 + 微光边框实现的现代 UI 效果，让卡片看起来像磨砂玻璃。

### TypeScript 接口

用 `interface` 定义数据的"形状"，告诉编辑器和编译器这个对象应该有哪些字段、每个字段是什么类型。写代码时有自动补全，运行前能发现类型错误。
